
# Sharpen filters

This filters has the ability to transform an image into a sharpen one, giving more definition to blur sections and soft edges

> :Formula align=center
>
> w = \begin{pmatrix}
> 0 & -1 & 0 `\\`
> -1 & 5 & -1 `\\` 
> 0 & -1 & 0 
> \end{pmatrix}
